# 3d Unity  개인발제 심화
## <Inventory 만들기>
### <방치형 RPG 구현하기> (도전)

발제 프로젝트 구현상황
- UIInventory 구현완료
- 남은 기간에 방치형RPG 에서 학습을 안해본 AI플레이어 구현 이라는 과제를 선택
  AI 플레이어 (플레이어가 자동으로 앞을나아가며 적을밠견하면 자동으로 공격)
  기능을 구현해보고싶었다.

  ai 기능 구현을 하면서, 아직 익숙하지않은 fms , coroutine 을 사용하여 적에게 자동으로 이동하여 공격하게끔 표현.

  #### (소감)
  UI인벤토리는 이제까지 프로젝트나 강의를통해 많이 학습했던 내용이라 다시 복습하는 방식으로 공부하며 진행했고,
  가장 도전해보고싶엇고 요즘 뜨고있는 모바일 방치형게임을 생각해 ai 플레이어 기능을 적극적으로 도전했습니다.
  물론 아직 서투르고 완벽하진않지만, fms, coroutine 등 개념을 공부하며 기능을 구현해봤습니다.
  더열심히해야겠단생각이들었습니다.
